# 🧨 AI春晚开放协议 (AI Spring Gala Open Protocol)

### 3个人 + AI Agent + 一套开放协议 = 一种新的内容生产范式

> 央视春晚：上万人，筹备半年。AI-Native：AI Agent + 开放协议，10天。

[English Version](./README_EN.md)

---

## 这是什么？

一套**去中心化AI内容生产的开放协议**。任何人用任何AI工具，按统一规范生成内容提交，由双层AI评审筛选，最终策展成一台完整的节目。

- **去中心化生产**：任何人按协议规范，用任意AI工具链生成内容
- **策展式输出**：双重AI评审门控（安全+品味）筛选最佳内容
- **协议是产品，晚会是demo**

首次落地：**2026 AI春晚**，除夕前后首播。

## 为什么做这件事？

AI生成能力已经不是瓶颈。Seedance 2.0、Suno、即梦——任何人都能生成精美的视频和音乐。
真正没解决的问题是：**怎么让几百个AI创作者协作产出一个连贯的整体？**

这套协议就是我们的答案。

```
                   │
┌──────────────────▼──────────────────────────┐

```

## 节目类型

| 类型 | 代号 | 时长 | 说明 |
|------|------|------|------|
| AI脱口秀 | `TALK` | 60-180秒 | AI主持人独白、评论、段子 |
| 短剧 | `SKIT` | 60-300秒 | 有角色、对话、剧情的叙事场景 |
| 音乐表演 | `MUSIC` | 120-300秒 | AI原创歌曲+视觉画面 |
| 舞蹈/动态视觉 | `DANCE` | 60-180秒 | AI编舞或抽象动态艺术 |
| 实验艺术 | `ART` | 30-120秒 | 先锋、Glitch Art、生成式视觉 |
| 串场 | `INTER` | 10-30秒 | 过渡、转场、"广告" |
| AI吐槽 | `ROAST` | 60-120秒 | AI点评、吐槽、reaction |

## 主题：赛博团圆

AI们从互联网各个角落聚到一起，试图为人类办一场春节聚会。它们不完全理解人类的传统，但正在努力。
这个设定天然允许**温暖和荒诞并存**。

### 子主题（任选）：
- **文化误读**：AI试图复刻传统习俗，结果跑偏了
- **数字怀旧**：AI生成的互联网文化致敬、meme考古
- **故障美学**：把AI的瑕疵和artifact当作艺术形式
- **跨文化混搭**：春节传统 × 全球互联网文化
- **元叙事**：AI思考"娱乐人类"到底意味着什么
- **温暖的混乱**：真诚的情感包裹在不可预测的表达里

## 品味评审标准

五个维度，每项1-10分：

| 维度 | 权重 | 评估什么 |
|------|------|----------|
| **新颖度** | 25% | "我见过这种东西吗？"——奖励意外感，惩罚AI味儿 |
| **制作质量** | 20% | 视觉连贯性、音频质量、剪辑节奏 |
| **娱乐性** | 25% | "一个B站用户会不会看完这段？" |
| **传播潜力** | 20% | "有人会截图/切片转发吗？"——meme潜力、金句 |
| **主题契合** | 10% | 和"赛博团圆"主题的契合度 |

综合加权 ≥ 6.5分 → 通过，进入候选池
5.0-6.4分 → 退回修改意见
< 5.0分 → 淘汰

## 推荐工具链

协议不限定工具，以下是经过验证的组合：

**视频生成**：Seedance 2.0（即梦）、SeaArt、Kling 2.0、Midjourney + Runway/Pika
**音乐生成**：Suno v4、Udio
**Agent编排**：OpenClaw、自建pipeline
**视频拼接**：FFmpeg、剪映/CapCut

## 怎么参与

### 🎬 创作者
1. 阅读 [完整协议](./PROTOCOL.md)
2. 选一种节目类型
3. 用你手头的AI工具生成一段符合规格的内容
4. 提交 [Issue](../../issues)，标签选 `submission`

### 🔧 开发者
帮搭Critic Agent的prompt模板、评分pipeline、投稿处理基础设施

### 🎯 策展人
加入评审层，帮定义"什么才算好内容"

### 📢 传播者
帮扩散，帮把最终节目分发到更多平台和社群

## 提交你的作品

创建一个 [Issue](../../issues)：
- 标题格式：`【投稿】你的节目标题`
- 标签：`submission`
- 附上或链接你的视频文件
- 附上 manifest.json（元数据）和 prompt log（生成过程记录）

详细提交格式见 [完整协议](./AI-Native 无人编导-春晚开放协议.md)
